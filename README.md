## 🛠 기술 스택 (Tech Stack)

- **Frontend**: React, TypeScript, Vite
- **Code Quality**: ESLint, Prettier
- **Editor**: CodeMirror
- **Code Highlight, Memo** : Yjs
- **State Management**: Redux
- **Styling**: SCSS
- **Real-time Communication**: WebSocket

## 설치 방법 (Installation)

```bash
# 저장소 클론
git clone https://github.com/PrograMemo-Groom/900pro-fe.git

# 디렉토리 이동
cd 900pro-fe

# 패키지 설치
npm install

# 개발 서버 실행
npm run dev
```
